1. What Is This Project?

This project builds a machine learning model that predicts how a person's body will process a drug based on their genetic variant. This area of science is called pharmacogenomics — the study of how genes affect drug response.

Core Question the Model Answers:
Given a patient's gene variant (e.g., CYP2D6*4/*1) and the drug they are taking (e.g., codeine), is this patient a Poor, Intermediate, Normal, or Ultrarapid metabolizer?

Knowing the metabolizer type matters clinically: a poor metabolizer cannot break down the drug efficiently, so it accumulates in the body and may cause toxicity. An ultrarapid metabolizer breaks down the drug too fast, making it ineffective.

2. The Two Genes Covered

GeneFull NameCommon Drugs Affected
CYP2D6Cytochrome P450 2D6Codeine, Tramadol, Tamoxifen, Antidepressants (SSRIs), Antipsychotics
CYP2C19Cytochrome P450 2C19Clopidogrel, Omeprazole, Antidepressants, Antifungals

Both genes are enzymes in the liver that metabolize roughly 25–30% of all prescription drugs.

3. Dataset

Source

PharmGKB / ClinPGx — a curated clinical pharmacogenomics database. File used: var_drug_ann.tsv (12,852 rows total). Only CYP2D6 and CYP2C19 rows with metabolizer labels were used.

How Raw Data Was Cleaned

Each row in the raw file describes a comparison study — it lists multiple alleles and assigns a metabolizer label to the overall group. To train a model, we needed one label per individual diplotype (*A/*B). A sentence parser was written to:

  1. Extract all *A/*B diplotype patterns from the sentence text.
  2. Split on phrases like "as compared to" to separate subject allele from comparison.
  3. Assign the metabolizer label only to the subject diplotype.
  4. Deduplicate by majority vote when the same diplotype appeared in multiple rows.
Result: 186 raw rows → 691 extracted diplotype-label pairs → 106 clean, deduplicated training examples saved as cleaned_data.csv.

4. Star Allele System & Activity Scores

Genes like CYP2D6 are reported using star allele notation: *1 is normal, *4 is non-functional, etc. A patient has two copies of each gene (one from each parent), so their genotype is written as a diplotype: *1/*4.

Each star allele has an activity score defined by CPIC (Clinical Pharmacogenomics Implementation Consortium):

Activity ScoreMeaningExample Alleles
0.0No function (null)CYP2D6*4, *5, *6
0.5Reduced functionCYP2D6*10, *41; CYP2C19*17
1.0Normal functionCYP2D6*1, *2; CYP2C19*1
1.5Increased functionSome CYP2C19 variants
2.0Gene duplicationCYP2D6*1x2 (2 functional copies)

The Activity Sum = score of allele 1 + score of allele 2. This maps directly to metabolizer class:

Activity SumMetabolizer Class
0.0Poor Metabolizer
0.5 – 1.0Intermediate Metabolizer
1.25 – 2.0Normal Metabolizer
> 2.0Ultrarapid Metabolizer

Features Used by the Model

FeatureTypeDescription
act_sumNumericActivity score sum (most important feature)
has_*4, has_*1, …Binary (0/1)Whether each common allele is present in the diplotype
drug_codeine, …Binary (0/1)One-hot encoding of the drug name

Model: Random Forest Classifier

  • 100 decision trees, each trained on a random subset of data and features.
  • class_weight="balanced" — compensates for unequal class sizes (few poor metabolizer examples vs. many normal).
  • Prediction = majority vote of all 100 trees.
  • Confidence = fraction of trees that agreed on the winning class.

Evaluation Results

MetricValue
Train Accuracy~0.980
Test Accuracy~0.857
5-Fold CV Accuracy~0.830 ± 0.04
Why is test accuracy lower than train accuracy?
Random Forests memorize training data well (hence ~98% train accuracy). Test accuracy (~86%) is the honest measure — the model sees 21 new examples it has never trained on. 5-fold cross-validation gives the most reliable estimate across all splits.

6. SHAP – Why Did the Model Predict That?

SHAP (SHapley Additive exPlanations) explains each individual prediction by showing how much each feature pushed the prediction toward or away from a class.

Example SHAP output for CYP2D6*4/*4 (Codeine):

Feature act_sum = 0.0 → pushes strongly toward Poor Metabolizer (+0.41)
Feature has_*4 = 1 → pushes toward Poor Metabolizer (+0.23)
Feature drug_codeine = 1 → small contribution (+0.05)

This gives the user (and clinician) a human-readable explanation of why the model made its prediction, not just a black-box answer.

7. The predict_metabolizer() API

This is the core function exposed to the frontend. It takes three inputs and returns a full prediction object.

Input Parameters

ParameterTypeExampleNotes
genestring"CYP2D6"Must be exactly "CYP2D6" or "CYP2C19"
variantstring"*4/*1"Diplotype in *A/*B format
drugstring"codeine"Lowercase drug name (see valid list below)

Output (Python dict → JSON)

{
  "prediction":     "poor metabolizer",
  "confidence":     0.92,
  "probabilities": {
    "poor metabolizer":        0.92,
    "intermediate metabolizer": 0.05,
    "normal metabolizer":       0.03,
    "ultrarapid metabolizer":   0.00
  },
  "activity_sum":   0.0,
  "alleles":        ["*4", "*1"]
}

Field Descriptions

FieldMeaning
predictionThe top predicted metabolizer class (one of four possible values)
confidenceFraction of Random Forest trees that agreed (0.0–1.0)
probabilitiesProbability for each of the four classes (all sum to 1.0)
activity_sumNumeric activity score total (biological basis of prediction)
allelesThe two parsed alleles from the variant string

Valid Metabolizer Classes (4 categories)

  • poor metabolizer — drug accumulates; risk of toxicity
  • intermediate metabolizer — slower than normal; may need dose adjustment
  • normal metabolizer — standard dosing applies
  • ultrarapid metabolizer — drug clears too fast; may need higher dose or alternative

Valid Gene Values

"CYP2D6" or "CYP2C19" (case-sensitive)

Valid Variant Format

Must be a diplotype string in the format *NUMBER/*NUMBER, e.g. *1/*1, *4/*2, *1x2/*1 (gene duplication).
Unknown alleles default to activity score 0.5 (reduced function assumed).

Common Valid Drug Names

All lowercase. Examples from the training data:

codeine  |  tramadol  |  tamoxifen  |  clopidogrel  |  omeprazole  |  fluoxetine  |  paroxetine  |  sertraline  |  risperidone  |  haloperidol  |  warfarin  |  venlafaxine  |  metoprolol  |  amitriptyline  |  nortriptyline

Edge case: If the frontend sends a drug not in the training data, the model will still run — the drug's one-hot column will be all zeros. The prediction will be driven entirely by the activity score and allele features. This is acceptable behavior.

Display Recommendations

  • Show prediction as the main result (large text, colored by class).
  • Show confidence as a percentage (e.g., 92%).
  • Show probabilities as a horizontal bar chart or progress bars for all four classes.
  • Show activity_sum with a tooltip explaining what it means.
  • Optionally show alleles to confirm what the backend parsed.

11. Quick Q&A Reference

Q: What does this tool predict?
Whether a patient is a poor, intermediate, normal, or ultrarapid metabolizer of a drug based on their CYP2D6 or CYP2C19 genetic variant.
Q: What machine learning model is used?
A Random Forest Classifier with 100 trees, balanced class weights, trained on 106 curated diplotype-drug-label examples.
Q: How accurate is it?
~86% on the held-out test set, ~83% on 5-fold cross-validation. Train accuracy is ~98% (the model memorizes training data well).
Q: Why is the training set only 106 rows?
Clinical pharmacogenomics data is sparse — only CYP2D6 and CYP2C19 rows with specific metabolizer labels were usable. 186 raw rows were expanded by sentence parsing into 691 pairs, then deduplicated down to 106 unique diplotype-drug combinations.
Q: What is SHAP?
SHapley Additive exPlanations — a method that shows how much each feature (activity score, alleles, drug) contributed to a specific prediction.
Q: What is the most important feature?
The activity sum (act_sum) — the numeric total of both alleles' activity scores. It encodes the biology directly and is the top feature by importance.
Q: Can the tool handle unknown alleles?
Yes. If an allele is not in the lookup table, the model defaults to 0.5 (reduced function). The prediction is still made but with lower confidence.
